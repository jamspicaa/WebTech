HTML

• language used to markup documents (i.e. web pages) in the WWW.

• historically, HTML was used to specify the structure and content of web pages, as well as their presentation.

• modern web development practice, however, uses HTML solely for semantic specification of web pages’ structure, and content, with presentational aspects being delegated to style sheets.

• application SGML (ISO 8879).

• initially developed at CERN, then the IETF, then the W3c, and then also the WHATWG.

• version history

 pre-standardization:

o HTML tags, by Tim Berners-Lee, late 1991.>

o HTML Internet Draft, by Tim Berners-Lee and Dan Conolly, published by the IEFT in mid-1993.

o HTML + (Hypertext Markup Format) Internet Draft, by Dave Raggett, late 1993.

 Standards:

o HTML 2.0 (RFC 1866, Nov. 1995).

 Supplement RFC’s:

• Form-based File Upload (RFC 1867, Nov. 1995).

• HTML Tables (RFC 1942, May 1996).

• Client side Image Maps (RFC 1980, Aug 1996).

• Internationalization (il8n) of HTML (RFC 2070, Jan 1997).

o HTML 3.2 (W3C Recommendation, Jan 1997)

o HTML 4 (W3C Recommendation, Dec 1997)

o HTML 4.01 (W3C Recommendation, Dec 1999)

• Introduced depreciation of presentational elements and attributes in favor of style sheets.

• Strict, Transitional, Frameset DTD’s.
