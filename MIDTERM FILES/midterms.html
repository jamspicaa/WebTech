    Ways to insert CSS

    There are three ways on how to insert a style sheet:

    1. External Style Sheet
      • When using an external style sheet, you can only edit just one file and the look of the entire website will change.
      • Each page must include a reference to the external style sheet file inside the <link> element. The <link> element goes inside the <head> section:
      • It can be created in any text editor and the style sheet must be saved with a .css file extension.
      • Take note that the file must not contain a html tags.

      Here is an example:

                          
                        <html>
                        <head>
                                <meta charset="UTF-8">
                                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                <title>Web Technology</title>
                                <link rel="shortcut icon" href="Images/Logo.png" type="image/png">
                                <link rel="stylesheet" type="text/css" href="style.css">

                        </head>
    2. Internal Style Sheet
      • Internal style sheet are used if there is one single page that have a unique style.
      • It is defined within the <style> element, inside the <head> section of an HTML page:

      Here is an example:

                          
                        <head>
                        <style>
                            body{
                                background-color: blue;
                            }

                            h1{
                                color:green;
                                margin-left: 20%;
                            }
                        </style>
                        </head>
    3. Inline Style
      • Inline style sheet are used to apply unique style for a single element.
      • To use inline styles, add the style attribute to the relevant element. The style attribute can contain any CSS property.

      Here is an example:

                          
                       <h1 style="color:green; margin-left:20%;">Lorem ipsum dolor sit amet.</h1>

    Cascading Order

    Whate style will be used when tehre are more than one style specified for an HTML element?

    As a rule we can state that all of the styles will "cascade" into a new "virtual" style sheet by the following rules, where number one has the highest priority:

    1. Inline style (inside an HTML element)
    2. External and internal style sheets (in the head section)
    3. Browser default

    So, an inline style (inside a specific HTML element) has the highest priority, which means that it will override a style defined inside the <head> tag, or in an external style sheet, or a browser default value.

      RGB Value

    • A color can be specified as an RGB in HTML by using this fomula:
      rgb(red, green, blue)
    • Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255.
    • For example, rgb(255, 0, 0) is displayed as blue, because blue is set to its highest value (255) and the others are set to 0.
    • To display the color black, all color parameters must be set to 0, like this: rgb(0, 0, 0).
    • To display the color white, all color parameters must be set to 255, like this: rgb(255, 255, 255).

                        text-align: center;
                        font-family:sans-serif;

                    <h1 style="background-color:rgb(0, 255, 0);">rgb(0, 255, 0)</h1>
                    <h1 style="background-color:rgb(0, 0, 255);">rgb(0, 0, 255)</h1>
                    <h1 style="background-color:rgb(248, 139, 234);">rgb(248, 139, 234)</h1>
                    <h1 style="background-color:rgb(139, 248, 214);">rgb(139, 248, 214)</h1>
                    <h1 style="background-color:rgb(251, 255, 52);">rgb(251, 255, 52)</h1>
                    <h1 style="color: rgb(255,255,255);background-color:rgb(0, 0, 0);">rgb(0, 0, 0)</h1>
                    <h1 style="border: 1px solid rgb(0, 0, 0);background-color:rgb(255, 255, 255);">rgb(255, 255, 255)</h1>

    • Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
    • Saturation is a percentage value, 0% means a shade of gray, and 100% is the full color.
      • Saturation can be describe as the intensity of a color.
      • 100% is pure color, no shades of gray
      • 50% is 50% gray, but you can still see the color.
      • 0% is completely gray, you can no longer see the color.

      • Here is an example:

                <style>

                        text-align: center;
                        font-family:sans-serif;

                <h1 style="background-color:hsl(0, 60%, 50%);">hsl(0, 60%, 50%)</h1>
                <h1 style="background-color:hsl(0, 40%, 50%);">hsl(0, 40%, 50%)</h1>
                <h1 style="background-color:hsl(0, 20%, 50%);">hsl(0, 20%, 50%)</h1>
                <h1 style="background-color:hsl(0, 0%, 50%);">hsl(0, 0%, 50%)</h1>

                <p>With HSL colors, less saturation mean less color. 0% is completely gray.</p>


                </body>
                </html>

  • Miscellaneous attributes and methods
    • nodeType
      • It returns a number of a specific node and returns a node type.
    • nodeName
      • It returns tha name of a specific node.
      • If the node is a element, the node will return a tag name.
      • If the node is a attribute, the node will return the name of the attribute.
    • nodeValue
      • It sets the node value of a specific node.
      • It will return to null if the node sets a element node.
  • Interface of a Node
    • Text Node
      • It creates a text node containing a specific text.
      • It uses the createElement() method to create an element node.
      • var t = document.createTextNode("Hello World");
    • Attribute
      • setAttribute
        • It adds a attribute into an element and provides the specified value.
        • The value will be changed if the specified value already exists.
      • getAttribute
        • It returns the corresponding value of the attribute with its corresponding name of the element.
