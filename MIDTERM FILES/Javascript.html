  • (Traditional) Browser Object Mode (BOM) API (non-standardized)
    • It allows the javascript interact to the browser
    • Window, Navigator, Screen, Location, History
  • Document Object Model (DOM) API
    • It is a API that allows access to the HTML document from within the scripts associated with the web page.
    • A parsed HTML document is represented by DOM TREE, which contains nodes.
    • It is accessed via DOCUMENT object property of the global window object.
    • Node, Document, Element, Text, Attribute, DocumentType, Comment
    • Event
      • It is used to provide contextual information about an event to the handler processing the event.
    • EventTarget
      • Represents an event target, and is a subset of the EventTarget interface defined in the DOM Level 3 Event model.
    • EventListener
      • Are defined which are invoked when a given event happens on a given object.
    • CSSStyleSheet, CSSRule
  • Miscellaneous JavaScript Web API’s
    • Geolocation, IndexedDB, Local Storage, Push Notifications, Service Workers, Web Sockets, Web Workers, XMLHTTPRequest (AJAX), etc.
    • Most API’s accessed
      • Window.navigator
      • It is an object that can be written without the window prefix.
    • Some examples:
      • navigator.appName
      • navigator.appCodeName
      • navigator.platForm
  • Different types of Linking the JavaScript to the HTML
    • Link
      • It is a set of data elements or nodes.
      • It can be measured as a high level standpoint where each node has atleast one pointer that represernts the nodes.
    • External
      • It uses the "src" attribute.
      • Used when a code is being used in multiple web pages.
    • Embedded (a.k.a. Internal)
      • It uses the "script" attribute.
      • It points to a external script and used to define a user side script.
  • Attributes
    • Defer
      • It downloads the script but it will parse the html.
      • It is only used for external scripts.
      • It tells whether a page is being parsed and if the script can be executed.
    • Async
      • The CSS and HTML are in parsing.
      • It is only used for external scripts.
      • It is a boolean attribute
      • It tells that the script is being executed asychronously as long at it is available.

    Functionalities

  • Relieve reference to nodes (or node collections) in the DOM
    • getElementByID()
      • It returns the ID attribute with its specified value.
      • It is used to manipulate and get information froma document.
      • It returns a null if there is no elements indicated in the ID attribute.
    • GetElementBytargetName()
      • It returns a NodeList Object that contains a collection of elements in a document specified in a name.
    • getElementByClassName()
      • It returns a NodeList Object that contains a collection of elements in a document specified in a class.
    • getElementByName
      • It returns a NodeList Object that contains a collection of elements in a document specified with name(value of the name attribute).
    • querySelector()
      • It returns the first element that have the same specified selector in a document.
    • querySelectorAll()
      • It returns the all element that have the same specified selector in a document.
  • Travers the DOM tree (from a give Node)
    • parentNode
      • It is a node object that returns the parent mode of a particular node.
    • parentElement
      • It returns the parent element of a particular element.
    • childNode
      • It returns a node of a child's node as a Nodelist object.
      • It is sorted according to its appeareance within the sourcecode.
    • childrenElement
      • It returns a collection of elements of a child's element as a HTMLCollection object.
      • It is sorted according to its appeareance within the sourcecode.
    • firstChild
      • It returns the first child node of a specific node as a Node object.
    • lastChild
      • It returns the last child node of a specific node as a Node object.
    • nextSibling
      • It returns the next specified node within the same tree level and returned as a Node object.
    • previousElementSibling
      • It returns the previous specified node within the same tree level and returned as a Node object.
  • Construct/Copy Nodes
    • createAttribute
      • It returns as a Attr object and creates an attribute with a specific name.
    • createTextNode
      • It creates a text node with a specific text.
    • createElement90
    • cloneNode()
      • It creates another copy of a node and clones all attributes with their corresponding values then returns the clone.
    • importNode()
      • It imports a specific node from another document and it can be imported to all kinds of nodes.
  • Manipulate the DOM TREE
    • appendChild()
      • It appends a node as a last child of the node.
    • insertBefore()
      • It inserts a node as a child before an existing child that is being specified.
    • replaceChild()
      • It replace a child node into a new node where the new node can be an existing node or a new node in a document.
    • removeChild()
      • It removes a child nodenode of a specified element and returns as a Node object or a null if it does not exist.
    • adoptNode()
      • It adopts a node in a document and that node can be all kinds of node..
  • Miscellaneous attributes and methods
    • nodeType
      • It returns a number of a specific node and returns a node type.
    • nodeName
      • It returns tha name of a specific node.
      • If the node is a element, the node will return a tag name.
      • If the node is a attribute, the node will return the name of the attribute.
    • nodeValue
      • It sets the node value of a specific node.
      • It will return to null if the node sets a element node.
  • Interface of a Node
    • Text Node
      • It creates a text node containing a specific text.
      • It uses the createElement() method to create an element node.
      • var t = document.createTextNode("Hello World");
    • Attribute
      • setAttribute
        • It adds a attribute into an element and provides the specified value.
        • The value will be changed if the specified value already exists.
      • getAttribute
        • It returns the corresponding value of the attribute with its corresponding name of the element.
